Indo-European keyboard
==============

Description
-----------
Keyboad for entering characters commonly used in Indo-European studies (Proto-Indo-European, Hittite, Sanskrit etc.)


## User guide

### Vowels

* Macron (ā): type vowel followed by \_. Also works with syllabic r̥ and vowel sign (V). 
* Breve (ă): type vowel followed by \\
* Acute accent (á): type vowel followed by /. This can be combined with long vowels (letter with macron)
* Grave accent (à): type vowel followed by `
* Slash after vowel: type / twice
* Semivowels (i̯ u̯): type vowel followed by <
* Lithuanian é: type e followed by .

### Consonants

* Laryngeals: type h followed by the number (1-3)
* Superscript h, w, i, j: type letter followed by ^
* Syllabic liquids (r̥, l̥, m̥, n̥): type letter followed by 0
* Retroflex consonants (ḥ etc.): type . followed by letter
* IE and Sanskrit palatal consonants (ḱ ǵ ś): type letter followed by /
* Palatal ñ: type n followed by ~
* Hittite ḫ: type h followed by \

### Other

* Zero grade marker (∅): type capital O followed by /
